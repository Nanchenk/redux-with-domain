## 背景

一个从蚂蚁数据平台孵化出的前端 SPA 框架，她专为开发复杂 web 应用而生。
在蚂蚁内部，我们喜欢叫她 KOP（King Of Prediction），而在外部，我们将其命名为 redux-with-domain，这样更能见名知意一些。

## 特性

1. 领域驱动设计

2. 状态分层
